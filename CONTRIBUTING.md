## Project status

This project is in Phase 0 (foundations). It does not yet build a usable artifact. Contributions during Phase 0 are limited to:

- Reviewing and commenting on the plan (`docs/00-README.md` → `docs/13-v1.1-backlog.md`).
- Opening RFC issues against still-open design decisions (see `docs/00-README.md`, "Decisions still open").
- Setup work explicitly listed in `docs/12-first-week.md`.

Module-level contributions open up after Phase 1 lands the locked interfaces (`docs/04-locked-interfaces.md`).

## Before you start

1. Read `docs/01-vision-and-scope.md`, `docs/02-architecture.md`, `docs/03-modules.md` (about 30 minutes).
2. Skim `docs/04-locked-interfaces.md` so you know what shape the contracts will take.
3. Check `docs/10-ownership-and-process.md` for the parallelization plan and module ownership.

## How to propose a change

- **Bug / feature / RFC / script proposal / template proposal:** open an issue using the matching template under `.github/ISSUE_TEMPLATE/`.
- **Pull request:** open against `main`. Keep PRs at module-boundary scope when possible; cross-module changes need a brief design note in the description.
